== Description ==

Bigboss CPT (Custom post type ) widget for displaying the latest posts of any post type including custom types. 
Control the widget title formatting and the number of posts to display. Plus the widget is completely localized for other languages.
If you face any problem then contact with me or see more about <a href="https://www.facebook.com/bigbosssoftbd/">bigboss</a> cpt post widget
